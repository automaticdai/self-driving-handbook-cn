# 第一章：概述

本章介绍自动驾驶的基本知识，包括自动驾驶的历史、定义、分类以及对未来智能交通的展望。

---

## 什么是自动驾驶？

自动驾驶（Autonomous Driving），又称无人驾驶或智能驾驶，是指车辆在无需人工干预或仅需极少干预的情况下，能够自主感知周围环境、规划行驶路线并安全完成驾驶任务的技术体系。它融合了人工智能、计算机视觉、传感器技术、高精地图、通信技术等多个前沿领域，被公认为是近年来最具变革性的技术方向之一。

自动驾驶技术的核心价值在于三个方面：

- **安全性**：全球每年有超过 130 万人死于交通事故，其中约 90% 由人为失误造成。自动驾驶系统不会疲劳、不会分心、反应速度远超人类，有望大幅降低事故率。
- **效率性**：自动驾驶车辆可以通过车联网协同行驶，减少拥堵、优化交通流，提升道路通行效率。
- **便利性**：老年人、残障人士等无法自主驾驶的群体将因此获得更大的出行自由。

## 各小节简介

### 1.1 定义（define）

本节对"自动驾驶"这一概念进行严格的技术定义。我们将探讨自动驾驶系统的边界条件——哪些场景属于自动驾驶、哪些属于辅助驾驶，并介绍构成一个完整自动驾驶系统的核心模块：感知、预测、规划与控制。

### 1.2 分类（category）

本节介绍目前最广泛采用的自动驾驶分级标准，包括 SAE International 发布的 L0 至 L5 六级分类体系，以及中国国家标准（GB/T 40429-2021）对应的分级定义。理解分级体系是把握行业现状与技术路线的基础。

### 1.3 历史（history）

本节梳理自动驾驶从概念提出到规模化落地的完整发展历程，包括 1980 年代德国 Ernst Dickmanns 的早期实验、DARPA 无人挑战赛的里程碑意义、谷歌/Waymo 项目的起步，以及国内外主流玩家近年来的商业化进展。

## 学习目标

读完本章后，读者应能够：

1. 用准确的语言解释什么是自动驾驶，并区分其与高级驾驶辅助系统（ADAS）的本质差异。
2. 熟练运用 SAE L0-L5 分级体系，能够判断具体产品属于哪个等级。
3. 了解自动驾驶技术的发展脉络，理解当前技术瓶颈的历史成因。
4. 建立对后续章节（系统、硬件、算法、案例）的整体认知框架。
